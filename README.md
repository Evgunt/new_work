# Проект обработки данных из AmoCRM и Google Sheets

Cкрипт подключается к API AmoCRM для получения событий за последние сутки, подсчитывает их типы, определяет топ-5 наиболее часто встречающихся событий и обновляет соответствующие данные в Google Sheets.

---

## Фаил events.php

- Получение списка пользователей из AmoCRM  
- Получение событий за последние 24 часа  
- Подсчет количества различных типов событий  
- Определение топ-5 самых частых событий  
- Перевод названий событий на русский язык  
- Обновление данных в Google Sheets  

---

## Фаил ping.php

- Выполняет GET-запрос к API AmoCRM для получения данных о сделках  
- Измеряет и отображает время выполнения запроса  
- Обновляет значения времени и текущей даты в Google Sheets  

---

## Требования

- PHP 7.4 или выше  
- Включённые расширения: `curl`, `json`, `mbstring`  
- Настроенные API-ключи и параметры доступа к **AmoCRM** и **Google Sheets**  
- Установка **Google API** через Composer
- Так же настройте параметры в файле `src/constants.php`
